OS/Dotfiles Plan

Install Steps:
    - Follow arch wiki almost completely
        - once chroot'ed run:
        # pacman -S dhcpcd
        - check for network interface, in my demo it was "enp3s0..." or something
        # ip link
        - start dhcpcd network service
        # systemctl enable dhcpcd@NETWORK_IERFACE
        
        OR
        # pacman -S NetworkManagern
        # systemctl enable NetworkManager
